## ūüď¶ Installation

Clone the repository:

```bash
git clone https://github.com/N1C1T0Z/creas-ai.git
cd creas-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run Creas :

On Windows :
```bash
start run-creas.bat latest
```

On Linux and MacOS :
```bash
bash run-creas.sh latest
```

## Run Folklorik :

On Windows :
```bash
start run-folklorik.bat dave-latest
```

On Linux and MacOS :
```bash
bash run-folklorik.sh dave-latest
```
