# 🤖 CREAS AI

**CREAS AI** is a next-generation digital AI assistant designed for creative thinking, intelligent interaction, and adaptive problem-solving.

Built to go beyond simple responses, CREAS AI aims to assist users in generating ideas, improving productivity, and creating meaningful digital experiences.

---

## 🚀 Features
* 🧠 Intelligent digital assistant capable of natural understanding and interaction
* 💡 Development-focused AI to help code, structure, and optimize projects
* 🎨 Creative idea generation (content, concepts, design, brainstorming)
* ⚡ Fast and adaptive responses based on user context
* 🔧 Modular and extensible architecture for customization and integration
* 🌐 Designed for multiple use cases: personal assistance, development, and professional projects

---

This project requires Python to run.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/N1C1T0Z/creas-ai.git
cd creas-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run Creas :

On Windows :
```bash
start run-creas.bat latest
```

On Linux and MacOS :
```bash
bash run-creas.sh latest
```

## Run Folklorik :

On Windows :
```bash
start run-folklorik dave-latest
```

On Linux and MacOS :
```bash
bash run-folklorik dave-latest
```
